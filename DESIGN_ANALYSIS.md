# Sorry Replica - Design Analysis

## Color Palette
- **Background**: Cream/beige gradient (oklch(0.999 0.0000455677 0.0000200868))
- **Primary Accent**: Soft pink (#FFB6C1 / #F8CEDC)
- **Secondary Accent**: Soft purple (#E6D5F8 / #D6D3DE)
- **Card Background**: White with subtle shadow
- **Text**: Dark brown/charcoal
- **Decorative**: Soft pink, purple, and cream gradients

## Typography
- **Heading Font**: Cursive/Script style (appears to be a handwriting font)
- **Body Font**: Sans-serif, clean and readable
- **Font Sizes**: Large headings, medium body text

## Layout Structure
- Centered card-based layout
- Full viewport height with centered content
- Floating decorative elements (hearts, shapes)
- Responsive design with mobile support

## Animation Details
- Floating hearts with gentle bounce
- Smooth fade transitions between steps
- Button hover effects with color changes
- Card entrance animations
- Decorative elements with subtle animations

## Steps Flow
1. **Step 1**: Initial greeting with character illustration
2. **Step 2**: Acknowledgment with character
3. **Step 3**: Message selection with 3 colored boxes
4. **Step 4**: Character illustrations with message preview
5. **Step 5**: Full message display with floral decorations

## Key Assets
- Character GIFs (holding hearts)
- Decorative hearts and shapes
- Floral elements
- Dashed border styling
